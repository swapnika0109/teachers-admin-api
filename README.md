# Teachers Admin API

A REST API for teachers to perform administrative functions for their students, built with Node.js and MySQL.

## Setup Instructions

1. Clone the repository.
2. Make sure Docker Desktop is running.
3. Run `docker compose up` to start the app and database.
4. Verify containers are running with `docker ps`.

The server will start at `http://localhost:3000`.

## API Swagger Documentation 

Once the server is running, visit `http://localhost:3000/api-docs` to view the interactive Swagger documentation.

## Running Tests

Run `npm test` to execute the Jest test suite.
